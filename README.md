# my_bashrc :smile::smile::smile:
My `.bashrc` configurations

# Features
- User@hostname
- Working Directory
- Git prompt
- Timestamp

# Setup
1. Copy `git-completion.bash` and `git-prompt.sh` file to your `~` directory.
2. Add content of `.bashrc` to the end of your `~/.bashrc`.
